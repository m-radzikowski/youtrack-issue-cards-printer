YouTrack Issue Cards Printer

Checkout the documentation.

Load default template (any custom content will be overridden).

Modify template or create your own. You may use default classes.

Use data-field attribute to link template element with data from YouTrack issue. Content is set inside child span element of node with data-field attribute.

Read about available fields.

Styles will be added as additional to the default card stylesheet.

Card structure:
.card-wrapper 8.8 x 8.8 cm
.card 7.8 x 7.8 cm
<card template>

In debug mode page with issue cards is opened, but without print popup right away.
It's useful when creating custom card layout.
